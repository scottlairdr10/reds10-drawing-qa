# Reds10 Drawing QA

AI-powered drawing QA review app for Reds10 Group.

## Deploy

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/scottlairdr10/reds10-drawing-qa.git
git push -u origin main
```

Then import the repo at vercel.com — no env vars needed.

## Local dev

```bash
npm install
npm run dev
```
